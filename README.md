## Environment
  - react-native-cli: 2.0.1
  - react-native: 0.50.3
  - npm: 5.6.0

### Dependencies Installation
```Bash
npm install
```

### Running the app for Android
```Bash
react-native run-android
```

### Running the app for iOS
```Bash
react-native run-ios
```
